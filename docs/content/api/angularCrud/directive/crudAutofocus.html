crudAutofocus

  1. - directive in module angularCrud

Helper directive to place focus on the first form field. If multiple fields are marked with this attribute, then focus will be placed on the first field.

Directive Info

  • This directive executes at priority level 0.

Usage

  • as attribute: 
    <input
  crud-autofocus="">
...
</input>

Arguments

Param Type Details
crudAutofocus expression

If the expression is truthy, then the control will gain focus when the view is ready.
