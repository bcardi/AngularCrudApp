crudEditableForm

  1. - directive in module angularCrud

Extends the x-editable form. Provides standard layout and actions.

Directive Info

  • This directive executes at priority level 0.

Usage

  • as element: 
    <crud-editable-form
  form-id=""
  crud-show-editable=""
  readonly="">
...
</crud-editable-form>

Arguments

Param Type Details
form-id string

Form ID.
crud-show-editable expression

If the expression is truthy, then the form will show in edit mode by default.
readonly expression

If the expression is truthy, then the form will display in read only mode and the user will not be allowed to make changes.
